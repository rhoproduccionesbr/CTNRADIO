# Guía de UX/UI: CTN Radio

## 1. Concepto Visual
- **Estilo:** Premium Dark / Glassmorphism.
- **Atributos:** Autoridad, Veracidad, Modernidad.

## 2. Paleta de Colores (HEX)
- **Fondo Primario:** `#0F0F0F` (Negro Profundo)
- **Acento Acción:** `#B71C1C` (Rojo Noticia)
- **Acento Secundario:** `#1A237E` (Azul Profundo)
- **Texto Principal:** `#FFFFFF` (Blanco)
- **Superficies (Cards):** `#1E1E1E` con opacidad del 80% (Glassmorphism).

## 3. Tipografía
- **Títulos:** `Montserrat` (Bold) - Para transmitir fuerza y seriedad.
- **Cuerpo:** `Lato` (Regular) - Para máxima legibilidad en noticias largas.

## 4. UX (User Experience)
- **Persistencia de Audio:** El reproductor no debe detenerse al cambiar entre secciones (noticias -> contacto).
- **Micro-interacciones:** - El botón de 'Play' debe tener un efecto de pulso suave cuando esté activo.
    - Las tarjetas de noticias deben tener un ligero escalado (hover effect) al pasar el cursor.
- **Mobile First:** Priorizar que el botón de escucha sea fácil de tocar con el pulgar en dispositivos móviles.

## 5. Accesibilidad
- Ratio de contraste mínimo de 4.5:1 para textos.
- Alt text descriptivo para todas las imágenes de noticias.
